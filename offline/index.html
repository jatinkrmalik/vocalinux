100% Offline

Offline Voice Dictation

Your voice never leaves your computer. All speech recognition happens locally using whisper.cpp, VOSK, or OpenAI Whisper models running on your hardware. No cloud. No upload. No exceptions.

No Internet Required

Works on airplanes, in tunnels, on remote farms. Your voice never needs to leave your machine.

Zero Data Collection

No telemetry, no analytics, no usage tracking. There's no server to send data to even if we wanted to.

No Account Required

No signup, no login, no password to forget. Download and run - that's it.

Your Voice Stays Yours

Voice data is biometric. Your voice patterns, accent, and speech habits are yours alone.

How Offline Processing Works

Vocalinux uses speech recognition models that run entirely on your CPU or GPU:

  • whisper.cpp - C++ port of OpenAI Whisper, runs locally with GPU acceleration
  • VOSK - Lightweight offline engine, minimal resources
  • OpenAI Whisper - PyTorch-based, runs on NVIDIA GPU or CPU

Models are downloaded once (74MB-3.0GB depending on size), then all processing happens locally. No network requests during dictation.

Verify It Yourself

Don't trust - verify. Vocalinux is open source. You can:

  • Read every line of code on GitHub
  • Run with --debug to see all network activity (there is none)
  • Monitor with tcpdump or wireshark - you'll see silence
  • Disconnect from the internet and verify dictation still works perfectly
